This is a list of things I use, inspired by uses.tech.

Hardware

  • Macbook Pro early 2015: buy Mac if you want to feel the heat!
  • OnePlus 7: Best mid-range Android phone, 2019.

Domain

  • Vercel: Vercel's domain search page is powerful and distraction-free, I only use it to quickly find out if a domain is available.
  • domcomp: Comparing the price of multiple domain registrars.
  • Namesilo: One of the cheapest domain registrar, free whois privacy service, supports refunds.
  • Porkbun: Sometimes even cheaper than Namesilo, free whois privacy service, doesn't support refunds.

Static Hosting

  • Vercel: I use Vercel to deploy static sites and serverless functions, it's free for personal use and you can get a free plan for open source teams.
  • Render: I use Render to deploy static sites when I need to collaborate with a team, it has free plan for closed source projects.
  • Netlify: I no longer use Netlify, their build minutes limit drived me crazy.

Backend Hosting

  • Vultr: Cheap, reliable, easy-to-use dashboard and it supports many payment methods.

Frontend Frameworks

  • React
  • Vue

Backend Frameworks

  • Prisma: Fluent API for relational databases.
  • TypeORM
  • Hasura: Generating production-ready GraphQL for your database, supporting role-base access control and custom business logic.

Development

  • Docker: used to spin up Postgres, Redis or some other libraies I don't want to install directly on my machine.
  • GitUp: a simple open-source Git GUI (I mainly use the CLI though).
  • VS Code: the best code editor so far.

Desktop Apps

  • Chrome: I might completely switch to Brave when this bug is fixed.
  • Telegram: my favorite messaging app.
  • WeChat: I just have to use it because everyone else in China uses it, it's awful.
  • Rocket: Typing emoji quickly.
  • IINA: A modern video player.
  • iStats Menus: Display memory usage and network stats on the menubar.
  • Itsycal: A tiny menubar calendar that works, it's open-source too!
  • Rectangle: Move and resize windows in macOS using keyboard shortcuts or snap areas.

Websites

  • AniList: Tracking my anime/manga progress.
